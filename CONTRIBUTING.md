# Contributing to Vanir Web

Thank you for considering contributing to the Vanir Web project! We welcome contributions from the community and are excited to collaborate with you. Please follow the guidelines below when contributing:

## Getting Started

1. **Fork the repository:**  
   Visit the [repository page](https://github.com/ahmed12345877/vanir-web) and click the "Fork" button in the top-right corner. This creates a personal copy of the repository in your own GitHub account.

2. **Clone the forked repository:**  
   In your terminal, run the following command to clone your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/vanir-web.git
   ```
   Replace `YOUR_USERNAME` with your GitHub username.

3. **Navigate into the repository:**  
   ```bash
   cd vanir-web
   ```

## Creating a Branch

Before making changes, create a new branch to avoid conflicts:
```bash
git checkout -b your-branch-name
```
Replace `your-branch-name` with a descriptive name for your branch that relates to the changes you plan to make.

## Making Commits

1. **Make your changes:**  
   Edit or add files as needed.

2. **Stage your changes:**  
   ```bash
   git add .
   ```  
   This stages all changes in the current directory. You can also specify individual files instead of `.` if needed.

3. **Commit your changes:**  
   ```bash
   git commit -m "A brief description of your changes"
   ```  
   Be sure to write a meaningful commit message that describes the changes you made.

## Submitting a Pull Request

1. **Push your changes to GitHub:**  
   ```bash
   git push origin your-branch-name
   ```

2. **Open a Pull Request:**  
   Go to the original repository on GitHub. You'll see a message about your recently pushed branch. Click "Compare & pull request".

3. **Fill out the Pull Request template:**  
   Describe your changes, why they are necessary, and any additional information that may aid the reviewers.

4. **Submit Pull Request:**  
   Click the "Create pull request" button to submit your changes for review.

## Code of Conduct

By participating in this project, you agree to abide by the [Code of Conduct](CODE_OF_CONDUCT.md).

## Thank You!

Thank you for your contributions! Together, we can make the Vanir Web project even better!
